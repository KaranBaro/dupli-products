# 🚀 Dupli Products

Duplicate Shopify products instantly with powerful bulk options.

---

## ✨ Features

- 🔁 Duplicate products in one click
- 📦 Bulk duplicate by:
  - Selected products
  - Tag
  - Product type
  - Collection
- 🖼 Copy images (optional)
- 🧬 Copy metafields (optional)
- 🏷 Add custom duplicate tags
- ⚡ Fast and optimized UI

---

## 🧠 How It Works

1. Open the app dashboard
2. Search product by title or SKU
3. Click **Duplicate** or **Bulk Duplicate**
4. Choose your settings
5. Click **Create duplicate**
